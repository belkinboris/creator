Редакция от 21 июля 2026 г.

Настоящее соглашение регулирует условия использования сервиса «Создатель» (projectsozdatel.ru), принадлежащего ИП Белкину Борису Ильичу (далее — «Администрация»). Используя Сервис, вы принимаете условия настоящего соглашения.

1. Что такое Создатель

«Создатель» — сервис для проверки предпринимательских идей. Он позволяет:

  • бесплатно оценить спрос на идею по данным поисковых запросов;
  • заказать «живой тест» — проверку готовности аудитории оставить заявку.

Сервис предоставляет данные и инструменты анализа; решения по идее принимает Пользователь самостоятельно.

2. Бесплатная проверка спроса

Проверка спроса доступна без регистрации и оплаты. Результаты основаны на публично доступных данных и оценках языковой модели. Сервис не гарантирует достоверность данных Яндекс Wordstat и поисковой выдачи — они предоставляются «как есть» третьими сторонами.

Результаты проверки сохраняются на 90 дней и доступны по уникальной ссылке.

3. Услуга «живой тест идеи»

Условия оказания платной услуги описаны в Публичной оферте. Оформление заказа означает согласие с офертой.

4. Обязательства Пользователя

Пользователь обязуется:

  • не вводить в Сервис идеи, связанные с незаконной деятельностью;
  • не использовать Сервис для автоматической массовой отправки запросов;
  • не предпринимать попыток обхода технических ограничений Сервиса;
  • не предоставлять заведомо ложные контактные данные при оформлении заказа.

  • временную недоступность Сервиса по техническим причинам.

Совокупная ответственность Администрации перед Пользователем не превышает стоимости оплаченной услуги.
