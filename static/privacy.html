ИП Белкин Борис Ильич (ИНН 771387918350, ОГРНИП 324774600432188), далее — «Оператор», настоящим информирует о порядке обработки персональных данных пользователей сервиса «Создатель» (projectsozdatel.ru).

1. Какие данные мы собираем

При проверке идеи (бесплатно): текст идеи, введённой Пользователем (до 300 символов). Данные не привязываются к личности без явного согласия.

При оформлении живого теста: контактные данные, добровольно введённые Пользователем (адрес электронной почты или Telegram-псевдоним) для связи по результатам теста.

При оплате: платёж проводится через ЮKassa. Номера банковских карт и платёжные данные Оператору не передаются и не хранятся.

Автоматически: IP-адрес и тип браузера для защиты от злоупотреблений (ограничение частоты запросов). Данные не используются для рекламного профилирования.

Мы не собираем паспортные данные, данные о местоположении, данные банковских карт.

2. Цели обработки

  • проведение проверки спроса и предоставление результата;
  • оказание услуги «живой тест идеи» и связь по её результатам;
  • проведение платежей через ЮKassa;
  • защита от злоупотреблений (rate limiting).

3. Правовые основания

Обработка персональных данных осуществляется на основании согласия Пользователя (ст. 6, п. 1 ФЗ-152), а также в целях исполнения договора (публичной оферты).

4. Передача третьим лицам

Данные передаются только в следующих случаях:

  • ЮKassa — для проведения платёжных операций;
  • Яндекс — тексты идей в обезличенном виде для оценки частотности через Wordstat и поиска конкурентов (без личных данных Пользователя);
  • Yandex Cloud / Anthropic — тексты идей для генерации предложений с помощью языковых моделей (без личных данных Пользователя);
  • Timeweb Cloud — хранение данных на серверах в России;
  • по требованию уполномоченных государственных органов РФ.

Мы не продаём данные третьим лицам.

5. Срок хранения

Результаты проверки идей хранятся 90 дней, после чего удаляются автоматически. Контактные данные, связанные с оплаченными заказами, хранятся 1 год для соблюдения требований законодательства.
